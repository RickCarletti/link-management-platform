# Link Management Platform

A full-stack link management platform featuring URL shortening, access analytics, authentication, and scalable cloud architecture.

---

## 🚀 Tech Stack

### Backend
- Node.js
- TypeScript
- Express
- Prisma ORM
- PostgreSQL

### Frontend (planned)
- React / Next.js

### DevOps / Infra
- Docker
- AWS / GCP (planned)

---

```
backend/
frontend/
docs/
```

## ⚙️ Setup Instructions

### 1. Clone the repository
```bash
git clone https://github.com/RickCarletti/link-management-platform.git
cd link-management-platform
```

### 2. Install dependencies
```bash
cd backend
npm install
```

### 3. Run the project
```bash
npm run dev
```

## 🌿 Git Workflow

### Branch naming
- `feat/...` → new features
- `fix/...` → bug fixes
- `chore/...` → setup / maintenance
- `test/...` → tests

### Example:
```
feat/url-shortener-endpoint
fix/login-validation
```

---

## 📝 Commit Pattern

We follow a conventional commit style:

```
feat: add URL shortening endpoint
fix: handle duplicate slug error
chore: configure eslint
test: add unit tests for link service
docs: update README
```

---

## 🔄 Development Flow

1. Create an issue
2. Create a branch from `main`
3. Implement the feature
4. Commit using standard pattern
5. Open a Pull Request
6. Merge after review
